ABOUT THIS PROJECT

Working of this system is mentioned in the steps below:
Working:
Step 1:
Patient uses a Web-API to provide all the required parameters to the model.
Step 2:
All the entered values of the parameters are forwarded to the model for prediction
Step 3: The trained model will be used to predict the probability of heart disease considering the input data provided by the individual.
Step 4: output will be displayed to the user.
About the Machine Learning model:
Model is trained on a dataset that contains 4240 tuples and 16 features on which we performed dimensionality reduction out of which we selected 11 features that are (Gender, age,current smoker, cigarettes per day, total cholesterol, systolic bp, diastolic bp, BMI, Heart rate,glucose levels) , which are relevant to the systemâ€™s requirement.
